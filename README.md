# King-gu

황새오래걷기 게임을 기반으로, 균형을 유지하며 300m를 이동하는 미니 게임입니다.

---

## 🎮 게임 소개

King-gu는 ‘황새오래걷기’ 게임을 레퍼런스로 만든 미니게임 프로젝트입니다.  
플레이어는 좌우 균형을 유지하며 캐릭터를 이동시켜 목표 거리인 300m에 도달해야 합니다.

단순한 조작이지만, 균형이 무너지면 진행이 어려워지기 때문에  
집중력과 타이밍이 중요한 게임입니다.

---

## ⚙️ 주요 기능

- 좌우 균형 기반 조작감
- 거리 누적 및 목표 도달 (300m)
- 균형 붕괴 시 실패 처리
- 간단한 입력 기반 조작
- 기본 UI를 통한 게임 상태 표시

---
